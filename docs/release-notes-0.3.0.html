  • Server configuration property for base NLP engine server.nlp.engine is removed and replaced by the system property NLPCRAFT_NLP_ENGINE. That system property can have two values: opennlp for Apache OpenNLP and stanford for Stanford CoreNLP engine. This system property must be set to the same value on both server and the probe JVMs. If not set the default NLP engine will be Apache OpenNLP.
